Volume Count
============

	An android application for data collection of vehicle traffic

Requirement
-----------
 Record the times a vehicle crosses a line on the road. To estimate the number of vehicles through the particular road

Funcitonality
------------

   The application has two main activities : Start sevice and stop service 

   Once the service is started it creates a file named by the current time to record data with a directory named by current date.
   Once a vehicle icon is clicked then the time it crossed the line is recorded and this data written into a file.
